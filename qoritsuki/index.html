Support сервер


Beta версия (unstable)

Команды:


/weather

Показывает текущую погоду. Параметры:


city

Обязательный параметр. Позволяет указать город, в котором нужно посмотреть погоду.

/video

Ищет видео из YouTube по запросу. Параметры:


query

Обязательный параметр. Позволяет указать название видео для поиска.

/translate

Позволяет переводить фразы на любой язык (который поддерживает Google Переводчик). Параметры:


query

Обязательный параметр. Позволяет указать фразу для перевода.


lang

Обязательный параметр. Позволяет указать язык, на который нужно перевести фразу.

/ping

Позволяет узнать задержку бота в миллисекундах. Параметров нет.

/changelog

Показывает последний changelog из файла changelog.md. Параметров нет.

/image

Ищет изображение из Google Images по запросу. Параметры:


query

Обязательный параметр. Позволяет указать изображение для поиска.

/dog

Гав! :)

/help

Показывает помощь по командам. Параметров нет.
