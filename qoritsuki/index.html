Документация 
/changelog

Показывает последний changelog из файла changelog.md. Параметров нет.

/delete

Удаляет выбранное количество последних сообщений. Параметры:


amount

Обязательный параметр. Позволяет указать количество сообщений, которые нужно удалить.

/dog

Гав! :)

/credits

Показывает, с помощью чего и кого был создан бот. Параметров нет.

/help

Показывает помощь по командам. Параметров нет.

/image

Ищет изображение из Google Images по запросу. Параметры:


query

Обязательный параметр. Позволяет указать изображение для поиска.

/math

Решает лёгкие математические выражения. Параметры:


expression

Обязательный параметр. Позволяет указать выражение для решения.

/status

Показывает статус бота (ОЗУ, CPU, задержка и др.). Параметров нет.

/translate

Позволяет переводить фразы на любой язык (который поддерживает Google Переводчик). Параметры:


query

Необязательный параметр. Позволяет указать фразу для перевода.


message_id

Необязательный параметр. Позволяет указать ID сообщения, которое нужно перевести.


lang

Необязательный параметр. Позволяет указать язык, на который нужно перевести фразу.

/video

Ищет видео из YouTube по запросу. Параметры:


query

Обязательный параметр. Позволяет указать название видео для поиска.

/weather

Показывает текущую погоду. Параметры:


city

Обязательный параметр. Позволяет указать город, в котором нужно посмотреть погоду.
